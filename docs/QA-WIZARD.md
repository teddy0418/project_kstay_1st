# 숙소등록 위저드 — 최종 QA 체크리스트 (비개발자용)

1. 호스트 로그인 후 `/host/listings` → 「새 숙소 등록」으로 위저드 진입.
2. 기본정보·위치·가격·어메니티·사진 입력 후 상단 「저장」 또는 각 스텝 「다음」으로만 저장됨. 상단에 "저장 필요" / "저장됨" 표시 확인.
3. 위치 스텝: 주소 검색 2자 이상 입력 시 "검색 중..." 표시 후 제안 목록 노출, 선택 시 주소·지도 반영.
4. 변경 후 다른 스텝 탭 클릭 시 "저장하지 않은 변경사항이 있습니다" 확인창 노출.
5. 검토 스텝: 제목·주소·가격·사진 1장 이상 없으면 「승인요청」 비활성. 모두 채운 뒤 「승인요청 (PENDING)」 클릭.
6. 판매관리: 「게스트 화면으로」, 초안/반려만 「삭제」 가능(확인 후 삭제).
7. 게스트 상세 페이지에서 어메니티·위치·사진 등 정상 노출 확인.
8. 로그아웃 상태에서는 주소 자동완성 검색 불가(호스트 전용).
