# 브랜드 로고 (결제/메인 노출용)

엑심베이에서 제공받은 **브랜드/결제 로고 파일**을 이 폴더에 넣어주세요.

## 사용 경로

이 폴더에 파일을 넣으면 웹에서 `/brands/파일명` 으로 접근 가능합니다.

## 권장 파일명 예시

- `visa.svg`
- `mastercard.svg`
- `jcb.svg`
- `amex.svg`
- `paypal.svg`
- `alipayhk.svg`
- `tng.svg` (Touch 'n Go)
- `dana.svg`
- `gcash.svg`
- `truemoney.svg`
- `paypay.svg`
- `econtext.svg`

## 형식

- SVG(선호) 또는 PNG
- 배경 투명 권장
